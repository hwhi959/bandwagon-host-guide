## 购买后第一步：找到你的 IP 和端口

买完之后，搬瓦工会发一封邮件到你的注册邮箱，里面有账户信息。找不到邮件也没关系，登录官方后台就能查到。

登录后台的路径是：

1. 打开搬瓦工官网，点右上角 **Client Area** 登录
2. 进去后点 **Services → My Services**，能看到你买的 VPS
3. 点对应 VPS 的 **KiwiVM Control Panel** 按钮，进入管理面板

KiwiVM 首页（Main Controls）就会直接显示你的 **Public IP address** 和 **SSH Port**。记下来，这两个是连接服务器必须的。

注意：搬瓦工的 SSH 端口不是默认的 22，是随机生成的，通常是一个五位数。

## 第二步：重置 Root 密码

买完之后默认是没有密码的，需要自己生成一次。

操作方法：
1. 在 KiwiVM 首页，先点 **Stop** 把 VPS 关机（密码重置需要 VPS 开机状态，这里说法有点反直觉——**要保持开机状态**，早期版本是关机，现在官方要求开机时重置）
2. 左侧菜单找到 **Root password modification**
3. 点 **Generate and set new root password**
4. 密码生成后**立刻记下来**。就显示这一次，刷新就没了

密码忘了怎么办？重新生成一次就好，这个操作可以反复做。

## 第三步：用 SSH 客户端连接服务器

现在有了三样东西：IP 地址、SSH 端口、root 密码。可以开始连了。

**Windows 用户**：推荐用 Termius 或者 FinalShell，都是免费的，界面友好，填进去 IP、端口、用户名（固定是 `root`，全小写）、密码就能连上。

**Mac 用户**：直接打开终端，输入：


ssh root@你的IP地址 -p 端口号


回车后输入密码，提示符变成 `root@xxxx:~#` 就说明连上了。

**Linux 用户**：同 Mac，直接 ssh 命令搞定。

连上之后，你就可以在这台服务器上干任何你想干的事了。

## KiwiVM 面板里还有哪些常用功能

讲真，KiwiVM 是搬瓦工自研的控制面板，同价位里算功能最全的，大多数常见操作都能自助完成，不用提工单。

**重装系统（Install new OS）**

左侧菜单点 Install new OS，选一个 Linux 发行版（Debian、Ubuntu、CentOS 都有，现在也支持 Ubuntu 26.04 了），勾选确认后点 Reload。重装前要先把 VPS 关机，不然会失败。重装完成后 SSH 密码会变，要重新生成一次。

**切换机房（Migrate to another DC）**

这是搬瓦工比较有特色的功能。买一个套餐，可以在后台一键切换到多个机房，不用重新买。切换后 IP 地址会变，如果有域名解析记得更新。

注意：如果 IP 已经被封，这个功能是用不了的，需要先花费更换 IP。

**快照与备份（Snapshots / Backups）**

免费的。快照是手动的，最多存 2 个，默认保留 30 天，可以设置为永久保存。备份是系统自动定期做的，剩余硬盘空间越多，备份频率越高。两个功能加起来，数据安全这块基本不用担心。

**Root Shell（在线命令行）**

不想装 SSH 客户端，可以直接在浏览器里操作。三种模式：basic（简单命令）、advanced（带日志输出）、interactive（接近本地终端体验）。应急用挺方便的。

## 几个新手容易踩的坑

**坑一：密码只出现一次**

Root 密码生成后只显示这一次，很多人没注意，直接关了页面。没关系，重新生成一次就行，这个操作没次数限制。

**坑二：SSH 连接超时**

搬瓦工的端口不是 22，是随机端口，不填端口直接连肯定超时。检查一下有没有填对 SSH Port。

**坑三：重装系统前没关机**

Install new OS 这一步需要先在 Main Controls 把 VPS 关机（点 Stop），再去点重装。顺序搞反了会报错。

**坑四：切换机房后域名解析没更新**

切换机房之后 IP 会变。如果有域名绑定在这台服务器上，切机房后记得去 DNS 更新 A 记录，不然网站会挂。
